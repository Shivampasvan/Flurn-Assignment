## Run Locally

Clone the project

```bash
  git clone https://github.com/uttamkrshaw/dirty-parcel-1754-Purplle-
```

Go to the project directory

```bash
  cd Backend
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run server (for backend)
  npm run start  (for frontend)
```
