 
# Project Title : PokeDex

## Run Locally

Clone the project

```bash
  git clone https://github.com/Shivampasvan/Flurn-Assignment
```

Go to the project directory

```bash
  cd flurn
```

Install dependencies

```bash
  npm install
```

```bash
  npm run start
```
